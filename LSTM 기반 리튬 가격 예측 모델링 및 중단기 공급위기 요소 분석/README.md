# 1. 배경 & 목적
---
+ 주제명
  + LSTM 기반 리튬 가격 예측 모델링 및 중단기 공급위기 요소 분석
+ 배경
  + 핵심 광물의 특정국에 대한 높은 의존도로 인한 공급 불확실성 확대
  + 전기차 수요와 공급만 문제로 인한 리튬 가격의 높은 변동성 발생
+ 목표
  + 리튬 가격 예측 모델을 개발하여 공급망의 리스크를 최소화하고, 정부 및 민간 기업에 수급 안정화 정보 제공
